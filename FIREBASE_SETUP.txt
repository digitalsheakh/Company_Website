🔥 FIREBASE SETUP INSTRUCTIONS

Your admin blog system is ready! Follow these steps to set up Firebase:

═══════════════════════════════════════════════════════════════

STEP 1: Create Firebase Project
═══════════════════════════════════════════════════════════════

1. Go to: https://console.firebase.google.com
2. Click "Add project"
3. Project name: "Digital Sheakh Blog"
4. Click Continue
5. Disable Google Analytics (optional)
6. Click "Create project"

═══════════════════════════════════════════════════════════════

STEP 2: Enable Authentication
═══════════════════════════════════════════════════════════════

1. In Firebase Console, click "Authentication" in left menu
2. Click "Get started"
3. Click "Email/Password" under Sign-in method
4. Enable "Email/Password"
5. Click "Save"

6. Go to "Users" tab
7. Click "Add user"
8. Email: digitalsheakh@gmail.com (or your email)
9. Password: (create a strong password)
10. Click "Add user"

═══════════════════════════════════════════════════════════════

STEP 3: Enable Firestore Database
═══════════════════════════════════════════════════════════════

1. Click "Firestore Database" in left menu
2. Click "Create database"
3. Select "Start in production mode"
4. Choose location: (closest to Bangladesh - asia-south1)
5. Click "Enable"

6. Go to "Rules" tab
7. Replace rules with:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /blogs/{document=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

8. Click "Publish"

═══════════════════════════════════════════════════════════════

STEP 4: Get Firebase Configuration
═══════════════════════════════════════════════════════════════

1. Click the gear icon (⚙️) next to "Project Overview"
2. Click "Project settings"
3. Scroll down to "Your apps"
4. Click the web icon (</>)
5. App nickname: "Digital Sheakh Website"
6. Click "Register app"
7. Copy the firebaseConfig object

It will look like this:

const firebaseConfig = {
  apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:xxxxxxxxxxxxx"
};

═══════════════════════════════════════════════════════════════

STEP 5: Add Configuration to Your Project
═══════════════════════════════════════════════════════════════

1. Create a file: .env.local in your project root
2. Add these lines (replace with YOUR values):

NEXT_PUBLIC_FIREBASE_API_KEY=AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your-project.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your-project-id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your-project.appspot.com
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=123456789
NEXT_PUBLIC_FIREBASE_APP_ID=1:123456789:web:xxxxxxxxxxxxx

3. Save the file

═══════════════════════════════════════════════════════════════

STEP 6: Test Your Admin Panel
═══════════════════════════════════════════════════════════════

1. Restart your development server:
   - Stop the current server (Ctrl+C)
   - Run: npm run dev

2. Go to: http://localhost:3000/admin

3. Login with:
   - Email: digitalsheakh@gmail.com (the one you created)
   - Password: (the password you set)

4. You should see the admin dashboard!

═══════════════════════════════════════════════════════════════

HOW TO USE THE ADMIN PANEL
═══════════════════════════════════════════════════════════════

✅ LOGIN:
   URL: https://sheakh.digital/admin
   
✅ DASHBOARD:
   - View all blog posts
   - See stats (total, published, drafts)
   - Delete posts
   
✅ CREATE NEW POST:
   - Click "+ New Blog Post"
   - Fill in title, excerpt, content (markdown supported)
   - Add tags (comma separated)
   - Add featured image URL (optional)
   - Click "Save as Draft" or "Publish Now"
   
✅ EDIT POST:
   - Click "Edit" on any post
   - Make changes
   - Click "Save" or "Publish"
   
✅ DELETE POST:
   - Click "Delete" on any post
   - Confirm deletion

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

❌ "Firebase: Error (auth/invalid-email)"
   → Check your email is correct

❌ "Firebase: Error (auth/wrong-password)"
   → Check your password is correct

❌ "Firebase: Error (auth/user-not-found)"
   → Create user in Firebase Console > Authentication > Users

❌ Can't see blog posts on /blog page
   → Make sure posts are marked as "Published"

❌ Environment variables not working
   → Restart development server after adding .env.local
