📝 WHERE ARE YOUR BLOGS STORED?

Your blog posts are stored in **Firebase Firestore Database** (Cloud Storage)

═══════════════════════════════════════════════════════════════

🗄️ STORAGE LOCATION
═══════════════════════════════════════════════════════════════

Database: Firebase Firestore
Collection Name: "blogs"
Location: Cloud (Google Firebase servers)

Each blog post is a document in the "blogs" collection with these fields:
- id (auto-generated)
- title
- excerpt
- content (markdown format)
- author
- tags (array)
- image (URL)
- published (true/false)
- date
- createdAt
- updatedAt

═══════════════════════════════════════════════════════════════

📍 HOW TO VIEW YOUR BLOGS
═══════════════════════════════════════════════════════════════

1. Go to: https://console.firebase.google.com
2. Select your project: "digital-sheakh-blog"
3. Click "Firestore Database" in left menu
4. You'll see the "blogs" collection
5. Click on it to see all your blog posts

═══════════════════════════════════════════════════════════════

✍️ HOW TO CREATE/EDIT BLOGS
═══════════════════════════════════════════════════════════════

Option 1: Admin Dashboard (Recommended)
- Go to: https://sheakh.digital/admin
- Login with your credentials
- Click "+ New Blog Post"
- Write and publish

Option 2: Firebase Console (Manual)
- Go to Firebase Console
- Navigate to Firestore Database
- Add/Edit documents manually

═══════════════════════════════════════════════════════════════

🔒 SECURITY
═══════════════════════════════════════════════════════════════

✅ Public can READ published blogs
✅ Only authenticated users can WRITE/EDIT/DELETE
✅ Drafts (published: false) are NOT visible to public
✅ All data is backed up by Firebase automatically

═══════════════════════════════════════════════════════════════

💾 BACKUP & EXPORT
═══════════════════════════════════════════════════════════════

Your blogs are automatically backed up by Firebase.

To export manually:
1. Go to Firebase Console
2. Firestore Database
3. Click "Import/Export" at top
4. Export to Google Cloud Storage

═══════════════════════════════════════════════════════════════

📊 BLOG DATA FLOW
═══════════════════════════════════════════════════════════════

1. You write blog in Admin Dashboard
2. Blog saved to Firebase Firestore
3. Blog appears on /blog page (if published)
4. Visitors read from Firebase in real-time
5. No need to rebuild or redeploy!
